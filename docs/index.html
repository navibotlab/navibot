Documentação do Navibot Platform

Introdução

O Navibot Platform é uma solução completa para gerenciar agentes de IA, permitindo a integração com diversos canais de comunicação, como WhatsApp. A plataforma oferece uma interface amigável para configurar e monitorar seus agentes de IA.

Com o Navibot Platform, você pode criar agentes personalizados, gerenciar arquivos, configurar armazenamentos de vetores e integrar com diferentes canais de comunicação.

Instalação

Para instalar o Navibot Platform, siga os passos abaixo:

Requisitos

  • Node.js v16 ou superior
  • npm v7 ou superior
  • PostgreSQL 12 ou superior
  • Conta na OpenAI e chave de API
  • Conta no Supabase (opcional, para armazenamento de arquivos)

Instalação Rápida

Linux/macOS

chmod +x install.sh
./install.sh

Windows

install.bat

O script de instalação irá guiá-lo através do processo de configuração, incluindo a criação do arquivo .env com as variáveis de ambiente necessárias.

Instalação Manual

# Clonar o repositório
git clone https://github.com/seu-usuario/navibot-platform.git
cd navibot-platform

# Instalar dependências
npm install

# Gerar Prisma Client
npx prisma generate

# Criar arquivo .env (veja a seção de Configuração)

# Aplicar migrações do banco de dados
npx prisma db push

# Construir o projeto
npm run build

Configuração

O Navibot Platform utiliza um arquivo .env para configurar as variáveis de ambiente. Abaixo estão as principais variáveis que você precisa configurar:

Variáveis de Ambiente

Variável Descrição Exemplo
DATABASE_URL URL de conexão com o banco de dados PostgreSQL postgresql://usuario:senha@localhost:5432/navibot
OPENAI_API_KEY Chave de API da OpenAI sk-abcdefghijklmnopqrstuvwxyz
NEXTAUTH_URL URL base para autenticação http://localhost:3000
NEXTAUTH_SECRET Chave secreta para autenticação sua_chave_secreta
SUPABASE_URL URL do Supabase (opcional) https://abcdefghijklm.supabase.co
SUPABASE_KEY Chave do Supabase (opcional) eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
Nota: O script de instalação irá criar automaticamente o arquivo .env e solicitar as informações necessárias.

Uso

Após a instalação e configuração, você pode iniciar o servidor de desenvolvimento ou de produção.

Modo de Desenvolvimento

npm run dev

O servidor de desenvolvimento será iniciado em http://localhost:3000.

Modo de Produção

npm run build
npm start

O servidor de produção será iniciado em http://localhost:3000.

Acesso ao Painel Administrativo

Acesse o painel administrativo em http://localhost:3000/admin.

Agentes de IA

Os agentes de IA são o coração do Navibot Platform. Eles são responsáveis por processar as mensagens recebidas e gerar respostas apropriadas.

Criação de Agentes

Para criar um novo agente, acesse o painel administrativo e clique em "Criar Agente". Preencha os campos necessários:

  • Nome: Nome do agente
  • Descrição: Descrição do agente
  • Modelo: Modelo de IA a ser utilizado
  • Instruções: Instruções para o agente
  • Funções: Funções disponíveis para o agente
  • Arquivos: Arquivos disponíveis para o agente

Edição de Agentes

Para editar um agente existente, acesse o painel administrativo, encontre o agente na lista e clique em "Editar". Você pode modificar todos os campos do agente.

Exclusão de Agentes

Para excluir um agente, acesse o painel administrativo, encontre o agente na lista e clique em "Excluir". Confirme a exclusão digitando "Delete" no campo de confirmação.

Atenção: A exclusão de um agente é permanente e não pode ser desfeita.

Gerenciamento de Arquivos

O Navibot Platform permite o gerenciamento de arquivos que podem ser utilizados pelos agentes de IA.

Upload de Arquivos

Para fazer upload de um arquivo, acesse o painel administrativo e vá para a seção "Arquivos de IA". Clique em "Adicionar Arquivo" e selecione o arquivo desejado.

Listagem de Arquivos

Na seção "Arquivos de IA", você pode ver todos os arquivos disponíveis, incluindo informações como nome, tamanho e data de upload.

Exclusão de Arquivos

Para excluir um arquivo, acesse a seção "Arquivos de IA", encontre o arquivo na lista e clique no ícone de lixeira. Confirme a exclusão digitando "Delete" no campo de confirmação.

Armazenamento de Vetores

O armazenamento de vetores permite que os agentes de IA acessem informações de forma mais eficiente, utilizando embeddings para buscar informações relevantes.

Criação de Armazenamento de Vetores

Para criar um novo armazenamento de vetores, acesse o painel administrativo e vá para a seção "Armazenamento de Vetores". Clique em "Criar Armazenamento" e preencha os campos necessários:

  • Nome: Nome do armazenamento
  • Descrição: Descrição do armazenamento
  • Arquivos: Arquivos a serem incluídos no armazenamento

Listagem de Armazenamentos de Vetores

Na seção "Armazenamento de Vetores", você pode ver todos os armazenamentos disponíveis, incluindo informações como nome, descrição e quantidade de arquivos.

Exclusão de Armazenamentos de Vetores

Para excluir um armazenamento de vetores, acesse a seção "Armazenamento de Vetores", encontre o armazenamento na lista e clique no ícone de lixeira. Confirme a exclusão digitando "Delete" no campo de confirmação.

Integrações

O Navibot Platform permite a integração com diversos canais de comunicação, como WhatsApp.

WhatsApp

Para integrar com o WhatsApp, acesse o painel administrativo e vá para a seção "Canais". Clique em "Adicionar Canal" e selecione "WhatsApp". Preencha os campos necessários:

  • Nome: Nome do canal
  • Agente: Agente de IA a ser utilizado
  • Tipo de Integração: WhatsApp Cloud API ou Z-API
  • Credenciais: Credenciais de acesso à API do WhatsApp

Outros Canais

O Navibot Platform está em constante evolução e novas integrações serão adicionadas no futuro.

API

O Navibot Platform oferece uma API para integração com outros sistemas.

Autenticação

A API utiliza autenticação JWT. Para obter um token de acesso, faça uma requisição POST para /api/auth/login com as credenciais de usuário.

Endpoints

A documentação completa da API está disponível em http://localhost:3000/api/docs.

FAQ

Como posso adicionar um novo usuário?

Atualmente, a adição de novos usuários é feita diretamente no banco de dados. Em versões futuras, será adicionada uma interface para gerenciamento de usuários.

Quais modelos de IA são suportados?

O Navibot Platform suporta todos os modelos disponíveis na API da OpenAI, incluindo GPT-3.5 e GPT-4.

Como posso personalizar o comportamento do meu agente?

Você pode personalizar o comportamento do seu agente através das instruções fornecidas durante a criação ou edição do agente. Além disso, você pode adicionar funções personalizadas e arquivos para melhorar a capacidade do agente.

O que fazer se eu encontrar um erro?

Se você encontrar um erro, verifique os logs do servidor para obter mais informações. Se o erro persistir, entre em contato com o suporte.

Dica: Para acessar esta documentação a qualquer momento, visite http://localhost:3000/docs quando o servidor estiver em execução.
